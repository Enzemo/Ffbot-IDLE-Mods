Installation Instructions

  1. Download the HTML files above and place them in the same folder as player_stats.json and game_stats.json.
  2. In OBS, add a new Browser Source.
  3. Check "Local file" and browse to the HTML file you want to use.
  4. Set the width and height to the OBS Dimensions listed above.
  5. (Optional) Set the "Refresh Browser when source becomes active" option if you want a clean start each time.
